## Objective
Learn basic Linux commands, file management, permissions, and system monitoring.

## Tasks Completed
1. Explored directories using `pwd`, `ls`, `cd`.
2. Created and deleted files/directories using `mkdir`, `touch`, `rm`, `rmdir`.
3. Viewed and edited files with `cat`, `less`, `nano`,`tail`.
4. Checked and modified permissions with `ls -l`, `chmod`, `chown`.
5. Monitored system resources using `top`, `htop`, `df -h`, `free -m`.


## Learning Outcomes
- Understand the Linux directory structure.
- Learn file permission management.
- Gain confidence navigating and managing Linux systems.
- Learn basic system monitoring commands.
